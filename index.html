    📚 Cómo Crear Archivos de Preguntas

    🗂️ Ubicación

    Crea archivos JavaScript en el directorio questions/

    Formato del nombre: questions_[TEMA].js

    📝 Estructura Básica

    // Array de preguntas para el cuestionario
const questions_mi_tema = [
    {
        question: "Tu pregunta aquí",
        type: "opcion", // "opcion", "varias" o "escribir"
        answers: ["A. Opción 1", "B. Opción 2", "C. Opción 3"],
        correct: 0 // índice de respuesta correcta
    }
];

    🎯 Tipos de Pregunta

    1. Opción Única (type: "opcion")

    {
    question: "¿Cuál es la capital de España?",
    type: "opcion", 
    answers: ["A. Madrid", "B. Barcelona"],
    correct: 0 // Madrid (índice 0)
}

    2. Múltiples Opciones (type: "varias")

    {
    question: "¿Cuáles son lenguajes de programación?",
    type: "varias",
    answers: ["A. JavaScript", "B. HTML", "C. Python"],
    correct: [0, 2] // JavaScript y Python
}

    3. Respuesta Escrita (type: "escribir")

    {
    question: "Comando para listar archivos en Linux:",
    type: "escribir",
    correct: "ls" // respuesta exacta
}

    ⚠️ Consejos Importantes

    • ✅ El nombre de la variable debe coincidir con el archivo
    • ✅ Usa comillas dobles para las cadenas
    • ✅ Los índices de array empiezan en 0
    • ✅ Escapa caracteres especiales: \"
    • ✅ Actualiza script.js para incluir tu archivo

    🔧 Último Paso

    Añade tu archivo a la lista en script.js:

    const allFiles = [
    // ... otros archivos
