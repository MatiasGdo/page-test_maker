# Página Web de Examen de Prueba

Este proyecto es una aplicación web diseñada para crear y gestionar exámenes de opción múltiple con fines de estudio. Los usuarios pueden agregar preguntas junto con sus respectivas respuestas, ver la lista de preguntas y gestionarlas de manera efectiva.

## Estructura del Proyecto

```
test-exam-webpage
├── src
│   ├── components
│   │   ├── AddQuestionForm.js
│   │   ├── QuestionList.js
│   │   └── QuestionItem.js
│   ├── styles
│   │   └── styles.css
│   ├── index.html
│   ├── index.js
│   └── App.js
├── package.json
├── .gitignore
└── README.md
```

## Características

- **Agregar Preguntas**: Los usuarios pueden agregar nuevas preguntas y sus respuestas correspondientes a través de un formulario.
- **Ver Preguntas**: Se muestra una lista de todas las preguntas agregadas, permitiendo a los usuarios revisarlas.
- **Editar/Eliminar Preguntas**: Los usuarios pueden editar o eliminar preguntas existentes según sea necesario.

## Comenzando

Para comenzar con este proyecto, sigue estos pasos:

1. **Clona el repositorio**:
   ```
   git clone <repository-url>
   ```

2. **Navega al directorio del proyecto**:
   ```
   cd test-exam-webpage
   ```

3. **Instala las dependencias**:
   ```
   npm install
   ```

4. **Ejecuta la aplicación**:
   ```
   npm start
   ```

5. **Abre tu navegador** y ve a `http://localhost:3000` para ver la aplicación.

## Uso

- Usa el formulario para agregar nuevas preguntas y respuestas.
- Revisa la lista de preguntas mostradas en la página.
- Edita o elimina preguntas según sea necesario.
